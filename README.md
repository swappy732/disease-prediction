# Disease Prediction System 🩺

## 📌 Overview
This project predicts diseases based on user symptoms using Machine Learning.

## 🚀 Features
- Predict diseases from symptoms
- ML models used (Random Forest, Decision Tree, etc.)
- User-friendly interface

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit / Flask

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
